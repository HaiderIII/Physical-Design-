## üìñ What is TCL?

**TCL** = **T**ool **C**ommand **L**anguage

- Scripting language created in 1988
- Used by EDA (Electronic Design Automation) tools
- Simple syntax: `command arg1 arg2 arg3`
- Everything is a string (flexible typing)

### Why TCL in Physical Design?
- OpenSTA uses TCL for scripting
- Synopsys, Cadence tools use TCL
- Industry standard for automation
